# Smart Machine Health Monitoring System

## 📌 Overview

The Smart Machine Health Monitoring System is an IoT-based project designed to monitor industrial machine conditions in real time using sensors. The system collects temperature, vibration, and sound data and displays it on a live dashboard.

This project demonstrates concepts of Industrial Automation and Predictive Maintenance.

---

## 🎯 Objectives

* Monitor machine parameters continuously
* Detect abnormal machine behavior
* Reduce unexpected machine failures
* Provide real-time monitoring dashboard

---

## 🛠️ Technologies Used

* Arduino / ESP32
* IoT Cloud (ThingSpeak)
* Sensors (Temperature, Vibration, Sound)
* Embedded C Programming
* Git & GitHub

---

## 🔧 Hardware Components

* Arduino UNO / ESP32
* DHT11 Temperature Sensor
* Vibration Sensor Module
* Sound Sensor Module
* WiFi Module (ESP8266)
* Breadboard
* Jumper Wires

---

## ⚙️ Working

1. Sensors collect machine parameters.
2. Microcontroller reads sensor data.
3. Data is transmitted to cloud platform.
4. Dashboard displays real-time values.
5. Alerts are generated when threshold limits are exceeded.

## 📊 Features

* Real-time monitoring
* Automatic dashboard updates
